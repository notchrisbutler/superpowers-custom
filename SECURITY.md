# Security Policy

## Supported Versions

SuperDuperPowers is currently in alpha. Security fixes target the current `main` branch until tagged releases are available.

---

## Reporting A Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Report security concerns privately to the maintainer with:

- A clear description of the issue
- Steps to reproduce when possible
- Affected harnesses or installation paths
- Any known impact or workaround

The maintainer will review the report and coordinate a fix or disclosure path as appropriate.
